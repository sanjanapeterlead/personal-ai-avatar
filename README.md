# Personal RAG Assistant

A local-first personal RAG chatbot built using:

- Python
- LlamaIndex
- Ollama
- Hugging Face Sentence Transformers

The application loads a personal documents like resume PDF, creates embeddings, builds a vector index and allows users to ask questions through a terminal-based chat interface.


## Features
- Local LLM using Ollama
- Local embeddings using HuggingFace
- Resume question answering
- PDF document support
- Retrieval Augmented Generation (RAG)

## Tech Stack
- Python
- LlamaIndex
- Ollama
- Sentence Transformers
